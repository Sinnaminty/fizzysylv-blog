Starting fizzysylv.xyz

I’ve been meaning to set up a proper personal site for a while, so I finally grabbed fizzysylv.xyz and decided to build it by hand.

Right now this site is just static HTML/CSS deployed through Cloudflare Pages. That’s intentional: I wanted something fast, simple, and easy to maintain while I focus on building cool projects instead of fiddling with a heavy stack.

What I’ll write about

  • Rust experiments (including my toy database project).
  • Notes from working on web apps and backend systems.
  • Crypto, security, and whatever I’m learning at the moment.
  • Life updates and miscellaneous “fizzy” thoughts.

If you somehow found your way here and want to say hi, you can always reach me via email on the main site.
